# newtouchOne主站前端

## 安装启动

``` sh
#安装
npm install

#开发启动
npm start

# 打包
npm run prod
```

###启动说明
- npm start 为开发环境，在浏览器中打开<http://localhost:5001>
- npm run prod 为生产环境，在浏览器中打开<http://localhost:8082>
- npm install安装失败可用cnpm install
